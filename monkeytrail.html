MONKEY TRAIL

The Monkey trail features technical rock gardens, boardwalks, jumps and drops. The trail finishes with the awesome ‘Lower Cliff’ descent. The start is reached by riding the first half of Follow the Dog Trail Monkey

Trail is a 6.5 mile moderately trafficked loop trail located near Keithville, Louisiana that features beautiful wild flowers and is rated as moderate. The trail offers a number of activity options and is accessible year-round. Dogs are also able to use this trail but must be kept on leash.
